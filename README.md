# Rigged-Coin-flip
This is a Vibe coded Coin flip app that has either fixed and random outcomes based on system volume

It work by fetching the volume from the system and using it to determine the next outcomer (heads or tails)
This is how it works in detail :
1. it will show random outcome if volume = 100%
2. it will how heads if volume is just one click below hundred ( 90%)
3. it will show tails if volume is anything below 1 click (<90%)

Note : it is an android app that works tatally fine in android 16 it might break in other conditions

it is made purely for entertainment and educational purposes,
and to spread awareness that how easily such things can be rigged against you.
